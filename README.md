# PiSynth

A diy synthesizer project for raspberry pi


## Dependencies
```
sudo apt install libasound2-dev jackd2
git clone https://github.com/thestk/rtmidi.git
cd rtmidi
mkdir build
cd build
cmake ..
make
sudo make install
```

## Building
cd to the root of the project and run:

```
mkdir build
cd build
cmake ..
make
```
